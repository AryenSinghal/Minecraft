# Minecraft
A simple recreation of Minecraft

Place blocks, delete blocks and change building materials.

## Requirements
- Ursina must be installed

## Instructions
- WASD to move
- Spacebar to jump
- Left click to place block
- Right click to delete block
- Number keys to switch materials:
  - 1 - Grass
  - 2 - Stone
  - 3 - Brick
  - 4 - Dirt
